# memeinsider-dl
A script to download [Meme Insider](https://memeinsider.co/) issues

# Installation
    git clone https://github.com/flomk/memeinsider-dl $HOME/.memeinsider-dl
    echo PATH="$HOME/.memeinsider-dl:${PATH}" >> ~/.zshrc
    source ~/.zshrc

# Usage

    memeinsider-dl -l
